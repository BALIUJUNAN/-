# 超市管理系统 Web 版

基于 Python Flask 的超市管理系统，可在 Windows 11 本地运行。

## 功能模块

- **销售收银**: 条码扫描、挂单、多种支付方式（现金/微信/支付宝/储值卡）
- **商品管理**: 商品录入、库存管理、批次管理
- **会员管理**: 会员注册、积分阶梯制度、等级调整
- **储值卡**: 储值卡创建、充值、消费、交易记录
- **促销管理**: 单品折扣、满减促销、第N件优惠、买M赠N
- **报表**: 销售报表、盈亏报告、导出CSV/HTML
- **采购管理**: 采购订单、入库审批
- **员工管理**: 员工账号、角色权限

## 积分阶梯制度

| 消费金额 | 积分比例 |
|---------|---------|
| 100元以下 | 1元 = 1积分 |
| 100-500元 | 1元 = 2积分 |
| 500-1000元 | 1元 = 3积分 |
| 1000元以上 | 1元 = 5积分 |

## 储值卡折扣

| 会员等级 | 储值卡折扣 |
|---------|----------|
| 普通会员 | 无折扣 |
| 银卡会员 | 2%折扣 |
| 金卡会员 | 5%折扣 |
| 钻石会员 | 10%折扣 |

## 运行方式

### 方式一：直接运行

```bash
# 进入 web 目录
cd web

# 安装依赖
pip install flask

# 运行服务器
python app.py
```

然后打开浏览器访问: http://127.0.0.1:5000

### 方式二：双击运行

直接双击 `start.bat` 文件即可启动
